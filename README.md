# Atrium

> A quiet space for students to share what they know.

Atrium is a private platform where university students share academic insight to help peers register for courses with confidence.

## The problem

Course registration is one of the most consequential decisions students make each semester, yet most of us make it without real information about teaching style, workload, or expectations. Conversations that would help — *"how does this professor structure exams?"*, *"how much reading is realistic per week?"* — happen in scattered WhatsApp threads and disappear within hours.

Atrium is a structured, private space for that knowledge to live and accumulate over time.

## What this is (and isn't)

Atrium is for **academic insight, not personal judgment**. Helpful contributions describe how a course was structured, what kind of preparation worked, and what advice might help students considering the course. Personal commentary, unsubstantiated claims, and anything outside academic experience are not welcome here.

## Status

In active development. This is the v0.1 technical core: a working REST API, a SQLite-backed insight system, and a minimal frontend.

## Tech stack

- **Backend:** Python, Flask, SQLite
- **Frontend:** HTML, CSS, vanilla JavaScript
- **Testing:** pytest
- **CI:** GitHub Actions
